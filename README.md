# Jenkins and Continuous Testing Assignment

## Project Overview
This project demonstrates Continuous Integration using Jenkins, Maven, and GitHub.

## Jenkins Installation
1. Install Java JDK 11+
2. Install Jenkins LTS
3. Access Jenkins at http://localhost:8080

## Jenkins Jobs
- Freestyle Job: SCM polling, batch command, Python script
- Multibranch Pipeline: Branch-based Jenkinsfile execution

## How to Run
- Push code to GitHub
- Jenkins automatically triggers builds

## Troubleshooting
- Ensure Git and Java paths are configured
- Check Console Output for errors
